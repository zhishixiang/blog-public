AList搭建教程&amp;网盘迁移至AList公告

July 30, 2022 · 时长: 2分钟 · 字数: 846字 · zsx | Suggest Changes

##前言

好久没看搭建的oneindex了，正当想传点资源上去时，发现oneindex又挂了，主页一片空白。不过oneindex本身的缺点确实很多：配置比较麻烦（虽然配置教程与alist差不多，但oneindex的callback api已经很久没人提供了）、页面不够美观~~（高情商：简单明了）~~、不够稳定、只支持挂载onedrive。思来想去，我决定将网盘程序更改为AList。

AList比起oneindex又有什么优点呢？其实真不少： 1.可以挂载阿里云盘、天翼云盘、和彩云这类不限速网盘，下载体验比oneindex单线程好很多（并不是所有人都会aria2下载） 2.支持直接从网页上传资源，无需客户端，临时上传小文件体验好了不少。 3.现在还在持续更新（oneindex现在归谁管都没确定下来）。

事不宜迟，开整。

##安装

在文档里官方给出了4种安装方式：

  • 一键脚本
  • docker
  • 手动安装
  • PaaS

由于我有自己的国内服务器和备案域名，于是便不考虑Paas，直接选择了docker。为什么不用一键脚本呢？因为下载是真tm慢（竟然不挂个阿里云盘作为下载源，差评）。

使用docker安装稳定版只需输入以下指令： docker run -d –restart=always -v /etc/alist:/opt/alist/data -p 5244:5244 –name=“alist” xhofe/alist:latest

安装完毕后输入docker logs alist，如果显示start server @ 0.0.0.0:5244，那么就算成功了，继续配置反向代理。我这边使用的是宝塔面板，因此需要进入面板设置：

点击网站配置，进入反向代理配置，点击添加反向代理。 ![QQ截图20220730103158.png][1]

填入以下信息： ![QQ截图20220730103255.png][2]

点击保存，如果配置正确那么就可以通过绑定的域名访问网站了。

![QQ截图20220730110330.png][3]

输入获得的密码后再进行一番配置，就大功告成了。

至此。本站的云盘顺利迁移至AList，也更换了新的域名。新域名为[drive.bnnet.com.cn][4]，以前的资源都还在里面，之后将使用速度更快的阿里云盘分享资源。
